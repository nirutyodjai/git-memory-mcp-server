## 🚀 **Quick Start - Get Running in 3 Steps**

### Step 1: Install
```bash
npm install -g git-memory-mcp-server
```

### Step 2: Configure
```bash
git-memory-mcp init
```

### Step 3: Launch
```bash
git-memory-mcp start --servers 100
```

**That's it! Your MCP server system is now running with 100 servers.**
